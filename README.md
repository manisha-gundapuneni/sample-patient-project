# Patient Reported Outcomes (PRO) API

A RESTful API for managing Patient Reported Outcomes - structured symptom and quality-of-life reports tied to patient treatment. Built with Laravel 11 and MySQL 8.

## Features

- **Patient Management**: Create and manage patient records with unique MRNs
- **Instrument Templates**: Define questionnaire templates with multiple question types
- **Submissions**: Track patient responses to instruments over time
- **Aggregated Summaries**: View statistical summaries of patient responses

## Technology Stack

- PHP 8.4+
- Laravel 11+
- MySQL 8+
- Docker & Docker Compose (for local development)

## Setup Instructions

### Prerequisites

- Docker and Docker Compose installed
- OR PHP 8.4+, Composer, MySQL 8+ (for local development without Docker)
